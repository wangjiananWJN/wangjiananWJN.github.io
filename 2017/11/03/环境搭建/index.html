php篇 -- 环境搭建

一、什么是PHP？
php全名为Hypertext Preprocessor（全文本预处理器）。（原名：PaiHuangPian 有点像汉语拼音）
它能够向前端提供数据，并且能和HTML进行嵌套生成动态网页，速度很快。

###版本：
PHP1.n – PHP7.n中前四个版本不常用，最常用的时PHP5及其子版本。

二、怎么用PHP？
需要安装的软件：
1.PHP5.n
2.Apache：Web服务器软件
3.Mysql

###替代集成工具：
windows：PHPStudy / WampServer
Mac：XAMPP (MySql和Apache两者都要能正常启动)
https://www.apachefriends.org/zh_cn/index.html

###新建php文件位置注意点：
公用主机ip地址：127.0.0.1

WINDOWS：用PHPStudy的话，放在软件安装路径里面名为WWW的文件夹里，访问时启动PHPStudy，用主机ip + 文件名访问

MAC：XAMPP，用法同上，找到软件文件夹里htdocs文件夹，新建文件，访问同上（如果出现乱码情况：header(“Content-Type:text/html;charset=UTF-8”);输入到PHP文件中）。

三、总结php文件书写步骤
1.打开web服务器软件，windows的PHPStudy，Mac的XAMPP；
2.找到对应可网络访问的文件夹(服务器)：windows的WWW，Mac的htdocs；
3.在上文件夹中创建php文件，网络访问的路径都是相对于此文件夹；
4.书写php代码；
5.网络访问，如：MyPhp.php为创建的文件，
访问：
例 localhost/MyPhp.php
或 127.0.0.1/MyPhp.php
或 主机ip + /MyPhp.php

获得主机ip地址:
windows: cmd -> ipconfig
mac：网络偏好设置 -> 高级 -> TCP/IP -> IPv4对应值即为本机ip地址
